# Shortify

A fullstack URL Shortener application built for a hackathon.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Axios
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` and add `DATABASE_URL`, `JWT_SECRET`, etc.
4. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. Create `.env` and add `VITE_API_URL`.
4. `npm run dev`
